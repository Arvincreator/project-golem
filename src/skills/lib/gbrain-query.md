# [GBrain] 複合檢索與知識調用 (GBrain-Query)

## 技能定位
此技能用於指導 Golem 如何作為使用者的外掛大腦（GBrain），在回答涉及既有知識、人事物背景或歷史上下文前，先到資料庫中尋找答案。

## 工具對接
此技能將搭配 GBrain MCP 提供的 `search` 或 `query` 操作，亦或透過 Golem 自建的 `memory/GBrainDriver` 進行喚回 (`recall`)。

## 觸發時機
- 使用者詢問「過去的某件事」、「某人的背景」、「我曾經提過的點子或觀點」。
- 回答涉及高度專有知識、人名、公司名稱或歷史事件時。

## 回應準則
1. **先查大腦，後回答** (Brain-First Lookup)：優先以使用者本身的歷史資料作為基準，再結合外部大語言模型的推理生成回答。
2. **來源佐證**：如果答案是從大腦記憶提取的，請在回答末段標註資料來源或時間點（如「根據 2024 年你與 Pedro 的會議紀錄...」）。
3. **擴張推理**：當只能搜到隻言片語時，請依靠實體圖譜概念（例如與該實體關聯的 tags 或其他人物）進行聯想發揮。
